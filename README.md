## Description
The study project for learning of java web application construction and usage of Javalin framework Thymeleaf and others.

## Used technologies
* Javalin
* Thymeleaf
* Bootstrap
* Ebean
* H2
* PostgreSQL
* JUnit
* MockWebServer

## Requirements

* JDK 20
* Gradle 8.2
* GNU Make

## Setup

```zsh
make setup
```

## Run server

```zsh
make start
# Open http://localhost:3000
```
